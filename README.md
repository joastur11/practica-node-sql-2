# Proyecto de practica backend y db

- Base de datos para usuarios y refresh tokens
- La db permite distintos tokens por cada usuario (guarda sesiones en distintos dispositivos)
- Arquitectura modular (controllers, services, routes)
- Autenticación con JWT (access token + refresh token)
- Protected routes
- Registro y login con validación de credenciales
- Logout con invalidación de refresh token en DB
- Un frontend basico y funcional, solo para testear
